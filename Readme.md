### 1. upgrade pip
```python -m pip install --upgrade pip```

### 2. install virtualenv
```pip install virtualenv```

### 3. create virtualenv
```python -m venv env```

>Note: if using pycharm just skip 2 and 3 

### 4. source env
```source env/bin/activate```

### 5. install grpc
```pip install -r requirements.txt```
### 6. Making proto file
```make proto```
